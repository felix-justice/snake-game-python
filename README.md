# Snake Game

A classic Snake game developed in Python as part of Oregon State University's CS162 course.

## Features

- Player-controlled snake movement
- Food collection and snake growth
- Collision detection
- Game-over logic

## Skills Demonstrated

- Python
- Object-Oriented Programming
- Game logic
- Collision detection
- Program organization

## Running the Project

1. Make sure Python is installed.
2. Open a terminal in the project folder.
3. Run the main Python file.
